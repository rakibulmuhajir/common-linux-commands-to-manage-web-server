Install Nano

sudo yum install nano

get file or directory size

sudo du -sh /var

zip a file or folder

zip -r[flag for folders] zip-file-name folder-to-be-zipped zip -r httpdocs.zip httpdocs

disable root login

Go to the following file. Edit it and set [PermitRootLogin] to no. if its commented out, remove # sign

PermitRootLogin no
